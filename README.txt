PRAVALLIKA GODAVARTHI — 3D PORTFOLIO
====================================

FILES
  index.html            -> the whole site (open this)
  assets/pravallika.png -> your profile photo

RUN LOCALLY
  Just double-click index.html, OR for best results run a tiny server:
    python -m http.server 8000
  then open http://localhost:8000

HOST IT (public, free) — pick ONE:

  1) Netlify (easiest)
     - Go to https://app.netlify.com/drop
     - Drag this WHOLE folder onto the page. Done. You get a live URL.

  2) Vercel
     - https://vercel.com  -> New Project -> upload/import this folder.

  3) GitHub Pages
     - Create a repo, upload index.html + the assets/ folder.
     - Settings -> Pages -> Branch: main / root -> Save.
     - Site goes live at https://<username>.github.io/<repo>/

IMPORTANT
  Keep index.html and the assets/ folder TOGETHER in the same place,
  or the photo won't load.

EDIT LATER
  All text is inside index.html in the STATIONS / SKILLS sections near the top
  of the <script>. Change text there. To swap the photo, replace
  assets/pravallika.png (keep the same name).
